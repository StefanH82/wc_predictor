# World Cup 2026 Company Predictor

## Deploy to Vercel (5 min)

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → import the repo
3. Leave all settings as default → Deploy
4. Copy the URL (e.g. https://wc-predictor.vercel.app)

## Embed in SharePoint

1. Edit your SharePoint page
2. Add a web part → search "Embed"
3. Paste your Vercel URL
4. Set height to 800px
5. Publish

## Admin PIN
Default: wc2026admin  
Change it in src/App.jsx — search for ADMIN_PIN

## Local dev
npm install
npm run dev
